# About pensco-dashboard

An element providing a Dashboard for the Pensco demo.

## Requiments

Install Node.js (I use Homebrew) then:

    npm install -g bower

## Dependencies

Download the element's dependencies:

    bower install

## Usage

We recommend that you use [Polyserve](https://github.com/PolymerLabs/polyserve) which you can install via:

    npm install -g polyserve

And you can run it via:

    polyserve -p 3000

Once running, you can checkout the demo at `http://localhost:3000/components/pensco-dashboard/demo`.
